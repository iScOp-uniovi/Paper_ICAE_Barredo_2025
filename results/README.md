# Results

## Raw data

Given the sheer size of the executions of all the experiments, a zip file is provided with the name `RAW_DATA.zip`.

### List of experiments

For easy access the htmls for each algorithm are provided.

The algorithms used for each experiment is as follows:

- ICAE2025: It uses the NSGA-II algorithm with 2, 4, 8 and 16 hosts.
- ICAE2025-IBEA-EXT-ALL-16: It uses the IBEA algorithm with 16 hosts.
- ICAE2025-MOACO-EXT-ALL-16: It uses the MOACO algorithm with 16 hosts.
- ICAE2025-SMPSO-EXT-ALL-16: It uses the SMPSO algorithm with 16 hosts.
- ICAE2025-SPEA2-EXT-ALL-16: It uses the SPEA2 algorithm with  16 hosts.
